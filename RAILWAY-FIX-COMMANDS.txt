   • npm install saw existing node_modules and skipped rebuild
   • App tried to load Windows binary on Linux → CRASH

✅ AFTER:
   • node_modules removed from git
   • Railway runs npm ci (clean install)
   • better-sqlite3 compiles natively for Linux
   • Correct ELF binary is created
   • App loads successfully
